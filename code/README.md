# 📊 Marketing ROI Suite (Meetmux Capstone Project)

## 🎯 Objective

Analyze a ₹6,00,000 marketing budget across Bangalore and identify high and low performing campaigns using data analysis techniques.

---

## 🧠 Key Analysis Performed

* ROI (Return on Investment) calculation for each campaign
* Z-score analysis to identify underperforming campaigns
* Correlation heatmap to understand relationships between variables
* Subplot dashboard visualization

---

## 📈 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

---

## 📊 Insights

* Google Ads (Search Campaign) shows the highest ROI
* Instagram Story Ads and Facebook Awareness campaigns show lower ROI
* Strong correlation observed between conversions and revenue

---

## 💡 Strategic Recommendation

Shift 25–30% of the budget from low-performing campaigns (Instagram Story Ads, Facebook Awareness)
to high-performing campaigns like Google Search and LinkedIn B2B Leads.

---

## 📁 Project Structure

* data/ → dataset
* code/ → analysis script
* output/ → dashboard image

---

## 🚀 Conclusion

This project demonstrates how data-driven decision making can optimize marketing spend and improve ROI through analytical techniques like Z-score and correlation analysis.
